🎬 AI Workflow Guide

AI 에이전트와 협업하는 개발자를 위한
컨텍스트 연속성 기반 프로젝트 관리 시스템

🤖 AI 친화적 🎬 영화 제작 방식 📚 완전한 문서화 🚀 즉시 사용 가능
GitHub에서 보기 NPM 설치

✨ 핵심 기능

📋

Story (스토리)

전체 프로젝트 방향성과 백로그를 관리하여 AI가 프로젝트의 큰 그림을 이해할 수 있게 합니다.

🎬

Continuity (콘티)

현재 작업 상태와 다음 작업을 명확히 하여 AI가 맥락을 잃지 않고 작업할 수 있게 합니다.

📚

Backup (백업)

완료된 작업 기록을 체계적으로 관리하여 작업 간 연결성을 보장합니다.

🚀 빠른 시작

# 1. 저장소 클론 git clone https://github.com/Woo7i/ai-workflow-guide.git cd ai-workflow-guide # 2. 프로젝트에 설치 ./install.sh /path/to/your-project # 3. AI 에이전트에게 요청 "현재 진행상황 및 백로그 분석해줘"
